# Use the image inside a prompt

Generate content by providing images as input through messages. The system can analyze and describe images using advanced AI models.

## Basic Image Generation
Use images as input to generate descriptions or analyze visual content by providing image URLs.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.imageBlock
        .limit(2)
        .generate.fromMany({
            messages: [
                {
                    content: "Describe these images",
                },
                {
                    type: "image",
                    content: "https://www.treadfirst.co.uk/wp-content/uploads/2023/05/riseBlog.jpg",
                },
            ],
        })
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        ImageBlock(limit: 2) {
          _additional {
            generate(
              groupedResult: {
                messages: [
                  {
                    content: "Describe these images"
                  }, 
                  {
                    type: "image", 
                    content: "https://www.treadfirst.co.uk/wp-content/uploads/2023/05/riseBlog.jpg"
                  }
                ], 
                options: {}
              }
            ) {
              error
              groupedResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          alt
          createdAt
          ext
          height
          mimeType
          modifiedAt
          order
          originalName
          path
          pathString
          remoteId
          size
          sourceId
          url
          width
        }
      }
    }
    ```
  </Tabs.Tab>
