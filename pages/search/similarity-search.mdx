# Similarity Search

Find similar content across text, images, and records based on their characteristics and patterns.

## Text Similarity
Search for text content with similar meaning or context.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.textBlock
        .similar
        .text("artificial intelligence", {
            certainty: 0.8
        })
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument(
          nearText: {
            concepts: ["resume"],
            certainty: 0.8
          }
        ) {
          _additional {
            certainty
            distance
          }
          createdAt
          html
          mimeType
          modifiedAt
          originalName
          path
          pathString
          remoteId
          sourceId
          subtitle
          description
          tags
          text
          title
          properties
          toc
          authors
        }
      }
    }
    ```
  </Tabs.Tab>

## Advanced Text Similarity
Search using multiple concepts and directional preferences.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.textBlock
        .similar
        .text(["AI", "machine learning"], {
            certainty: 0.9,
            moveTo: {
                concepts: ["future"],
                force: 0.5
            }
        })
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
    ```
