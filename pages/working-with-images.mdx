# Working with images

Use Unbody's Image API powered by Imgix to efficiently fetch, transform and analyze images. Capabilities include dynamic resizing, format conversion, advanced cropping, and intelligent image analysis.

## Basic Image Query
The code fetches image information including OCR text extraction and AI-generated captions, limiting results to 5 entries.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const {
      data: { payload },
    } = await unbody.get.imageBlock
        .limit(5)
        .select("originalName", "autoOCR", "autoCaption")
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        ImageBlock(limit: 5) {
          originalName
          autoOCR
          autoCaption
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Similar Image Search
Find similar images by providing a reference image URL.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const {
      data: { payload },
    } = await unbody.get.imageBlock.similar
          .image(
              "https://www.treadfirst.co.uk/wp-content/uploads/2023/05/riseBlog.jpg" // Image about an electric car
          )
          .limit(5)
          .select("originalName", "url", "autoOCR", "mimeType")
          .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        ImageBlock(
          nearImage: {
            image: "https://www.treadfirst.co.uk/wp-content/uploads/2023/05/riseBlog.jpg"
          }
          limit: 5
        ) {
          _additional {
            certainty
            distance
          }
          originalName
          url
          autoOCR
          mimeType
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>
