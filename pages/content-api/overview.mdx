# Content API Overview

The **Content API** lets you interact with enriched, AI-ready data in Unbody. Built on [**Weaviate**](https://weaviate.io/), it extends core capabilities with additional data types and functionalities, allowing you to build intelligent, semantic applications with ease.

## What Can You Do with the Content API?

- Perform traditional database operations: [filter](/content-api/filters), sort, group, and limit results.
- Run [semantic searches](/content-api/search/semantic-search) using vector similarity and contextual queries.
- Execute [generative](/content-api/generative) tasks such as metadata creation, text synthesis, and data augmentation.
- Work seamlessly through GraphQL or a JavaScript SDK.

## How to Start?

1. **Choose Your Interaction Method**:
- Use the **GraphQL Interface** for precise query operations.
- Use the **JavaScript SDK** for streamlined integration into web applications.
2. **Authenticate**: Follow the [authentication guide](/content-api/authentication) to connect to the API.
3. **Explore Features**: Dive into specific sections to learn about advanced capabilities and examples.
