# Simple Rag With Generative Search

Use AI to generate meaningful insights and summaries from your documents using Retrieval-Augmented Generation (RAG). Apply natural language prompts to individual documents or across multiple sources to extract relevant information and create context-aware responses.

## Generate from single document
Let's say you have a PDF document and want to create a quick summary of its contents. This example shows how to generate an AI response that references any part of the document.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const {
      data: { payload },
    }  = await unbody.get.textDocument
          .where({ mimeType: "application/pdf" })
          .generate
          .fromOne("Summarize this: {text}")
          .select("title", "originalName")
          .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument(
          where: {
            operator: Equal, 
            path: "mimeType", 
            valueText: "application/pdf"
          }
        ) {
          _additional {
            generate(
              singleResult: {
                prompt: "Summarize this: {text}"
              }
            ) {
              error
              singleResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          title
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Generate from single document with options
Need more control over how the AI processes your documents? You can fine-tune parameters like creativity level and response length to get exactly the kind of output you need.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const {
      data: { payload },
    }   = await unbody.get.textDocument
            .where({ mimeType: "text/markdown" })
            .generate.fromOne({
                prompt: "Create a short report for these: {text}",
                options: {
                    model: "gpt-4",
                    topP: 0.7,
                    maxTokens: 1000,
                    temperature: 0.7,
                    presencePenalty: 0,
                    frequencyPenalty: 0,
                },
            })
            .select("originalName")
            .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument(
          where: {
            operator: Equal, 
            path: "mimeType", 
            valueText: "text/markdown"
          }
        ) {
          _additional {
            generate(
              singleResult: {
                prompt: "Create a short report for these: {text}", 
                options: {
                  model: "gpt-4", 
                  topP: 0.7, 
                  maxTokens: 1000, 
                  temperature: 0.7, 
                  presencePenalty: 0, 
                  frequencyPenalty: 0
                }
              }
            ) {
              error
              singleResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Generate from multiple documents
Have information spread across several documents? You can combine specific fields from multiple sources to create a unified summary or analysis.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const {
      data: { payload },
    } = await unbody.get.textDocument
          .generate
          .fromMany("Create a 100 words summary:", ["text", "title"])
          .select("title", "originalName")
          .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument {
          _additional {
            generate(
              groupedResult: {
                task: "Create a 100 words summary:", 
                properties: ["text", "title"]
              }
            ) {
              error
              groupedResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          title
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Generate from multiple documents with options
