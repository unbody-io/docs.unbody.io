# Basic using generative search

Use AI to generate meaningful insights and summaries from your documents by applying natural language prompts to individual documents or across multiple documents.

## Generate from single document
Transform a single document into an AI-generated response by providing a prompt that can reference any of the document's properties using curly braces notation.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.textDocument
        .where({ mimeType: "application/pdf" })
        .generate
        .fromOne("Summarize this: {text}")
        .select("title", "originalName")
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument(
          where: {
            operator: Equal, 
            path: "mimeType", 
            valueText: "application/pdf"
          }
        ) {
          _additional {
            generate(
              singleResult: {
                prompt: "Summarize this: {text}"
              }
            ) {
              error
              singleResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          title
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

```typescript
// Response
[
  {
    "_additional": {
      "generate": {
        "error": null,
        "metadata": {
          "finishReason": "",
          "usage": {
            "inputTokens": 5353,

          }
        },
        "singleResult": "This research paper, published in the International Journal of Innovative Technology and Exploring Engineering, explores the relationship between machine learning and deep learning, two branches of artificial intelligence that are integral to data science..."
      }
    },

```

## Generate from single document with options
Generate content from a single document with additional configuration options like model selection and generation parameters.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.textDocument
        .where({ mimeType: "text/markdown" })
        .generate.fromOne({
            prompt: "Create a short report for these: {text}",
            options: {
                model: "gpt-4",
                topP: 0.7,
                maxTokens: 1000,
                temperature: 0.7,
                presencePenalty: 0,
                frequencyPenalty: 0,
            },
        })
        .select("originalName")
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument(
          where: {
            operator: Equal, 
            path: "mimeType", 
            valueText: "text/markdown"
          }
        ) {
          _additional {
            generate(
              singleResult: {
                prompt: "Create a short report for these: {text}", 
                options: {
                  model: "gpt-4", 
                  topP: 0.7, 
                  maxTokens: 1000, 
                  temperature: 0.7, 
                  presencePenalty: 0, 
                  frequencyPenalty: 0
                }
              }
            ) {
              error
              singleResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Generate from multiple documents
Create a unified AI-generated response from multiple documents by specifying which properties to include in the generation process.

<Tabs items={['Unbody SDK', 'GraphQL']}>
  <Tabs.Tab>
    ```typescript
    const data = await unbody.get.textDocument
        .generate
        .fromMany("Create a 100 words summary:", ["text", "title"])
        .select("title", "originalName")
        .exec();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```graphql
    {
      Get {
        TextDocument {
          _additional {
            generate(
              groupedResult: {
                task: "Create a 100 words summary:", 
                properties: ["text", "title"]
              }
            ) {
              error
              groupedResult
              metadata {
                finishReason
                usage {
                  inputTokens
                  outputTokens
                  totalTokens
                }
              }
            }
          }
          title
          originalName
        }
      }
    }
    ```
  </Tabs.Tab>
</Tabs>

```typescript
// Response
[
  {
    "_additional": {
      "generate": {
        "error": null,
        "groupedResult": "Here is a summary of the provided text:\n\nThe text includes a variety of topics...",
        "metadata": {
          "finishReason": "",
          "usage": {
            "inputTokens": 14293,
            "outputTokens": 322,
            "totalTokens": 14615
          }
        }
```

