Generate human-like text content using natural language prompts or structured message sequences. Perfect for creating reports, summaries, or engaging in interactive conversations.

## Generation with Options
Control the output by specifying generation parameters like model, temperature, and token limits.

```typescript
const data = await unbody.generate
    .text("Write a report on climate change.", {
        model: "gpt-4",
        topP: 0.7,
        maxTokens: 1000,
        temperature: 0.7,
        presencePenalty: 0,
        frequencyPenalty: 0,
    });
```

## Multi-Message Generation
Create more contextual and guided responses using a sequence of system and user messages.

```typescript
const data = await unbody.generate
    .text(
        [
            { 
                role: "system", 
                content: "You are an expert on AI." 
            },
            { 
                role: "user", 
                content: "What are the latest AI trends in 2024?" 
            },
        ],
        {
            model: "gpt-4",
            topP: 0.7,
            maxTokens: 1000,
            temperature: 0.7,
            presencePenalty: 0,
            frequencyPenalty: 0,
        }
    );
```
